**[Internet Draft hoffman-xml2rfc: The 'XML2RFC' version 3 Vocabulary](/specs/IETF/I-D/hoffman-xml2rfc " This document defines the &#34;XML2RFC&#34; version 3 vocabulary; an XML-based language used for writing RFCs and Internet-Drafts. It is heavily derived from the version 2 vocabulary that is also under discussion. This document obsoletes the v2 grammar described in RFC 2629 and its expected followup, draft-reschke-xml2rfc."):** [This document updates the specification for the Internet media type "application/rfc+xml" from the one in I-D draft-reschke-xml2rfc.  The following is to be registered with IANA.](https://tools.ietf.org/html/draft-hoffman-xml2rfc-15#section-6.1)
