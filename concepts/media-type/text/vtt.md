[**W3C TR http://www.w3.org/TR/webvtt**: Web Video Text Tracks (WebVTT)](/specs/W3C/TR/webvtt "This specification defines WebVTT, the Web Video Text Tracks format. Its main use is for marking up external text track resources in connection with the HTML <track> element. WebVTT files provide captions or subtitles for video content, and also text video descriptions, chapters for content navigation, and more generally any form of metadata that is time-aligned with audio or video content."): []()
