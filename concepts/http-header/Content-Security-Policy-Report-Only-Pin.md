[**W3C TR http://www.w3.org/TR/csp-pinning**: Content Security Policy Pinning](/specs/W3C/TR/csp-pinning "This document defines a new HTTP header that allows authors to instruct user agents to remember (&#34;pin&#34;) and enforce a Content Security Policy for a set of hosts for a period of time."): [The Content-Security-Policy-Report-Only-Pin header field is the mechanism for delivering a pinned policy that the user agent MUST monitor for any resource which is not delivered with a Content-Security-Policy-Report-Only header (as described in the "Pin a policy to response" algorithm).]()
