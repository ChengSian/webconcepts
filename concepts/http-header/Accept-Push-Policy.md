**[Internet Draft ruellan-http-accept-push-policy: Accept-Push-Policy Header Field](/specs/IETF/I-D/ruellan-http-accept-push-policy "The &#34;Accept-Push-Policy&#34; and &#34;Push-Policy&#34; header fields enable a client and a server to negotiate the behaviour of the server regarding the usage of push on a per-request basis."):** [A client can express the desired push policy for a request by sending an "Accept-Push-Policy" header field in the request. The header field value contains the push policy that the client expects the server to use when processing the request.](http://tools.ietf.org/html/draft-ruellan-http-accept-push-policy#section-3.1)
