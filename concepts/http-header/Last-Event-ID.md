**[W3C TR http://www.w3.org/TR/eventsource: Server-Sent Events](/specs/W3C/TR/eventsource " specification defines an API for opening an HTTP connection for receiving push notifications from a server in the form of DOM events. The API is designed such that it can be extended to work with other push notification schemes such as Push SMS."):** [The Last-Event-ID HTTP header specifies the value of the event source's last event ID string, encoded as UTF-8.](http://www.w3.org/TR/eventsource/#last-event-id)
