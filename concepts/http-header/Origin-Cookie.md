**[Internet Draft west-origin-cookies: Origin Cookies](/specs/IETF/I-D/west-origin-cookies "This document updates RFC 6265, defining the &#34;origin&#34; attribute for cookies and the &#34;Origin-Cookie&#34; header field, which together allow servers to choose to harmonize the security policy of their cookies with the same-origin policy which governs other available client-side storage mechanisms."):** [The user agent includes stored cookies whose "origin-flag" is set in the "Origin-Cookie" request header. When the user agent generates an HTTP request, it MUST NOT attach more than one "Origin-Cookie" header field.](http://tools.ietf.org/html/draft-west-origin-cookies#section-4.4)
