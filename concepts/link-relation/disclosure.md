[**RFC 6579**: The 'disclosure' Link Relation Type](/specs/IETF/RFC/6579 "This document specifies the 'disclosure' link relation type. It designates a list of IPR disclosures made with respect to the material for which such a relation type is specified."): [Whenever the 'disclosure' relation type is used, the resource at the target Internationalized Resource Identifier (IRI) MUST represent a list of patent disclosures made with respect to the material referenced by context IRI.](http://tools.ietf.org/html/rfc6579#section-2)
