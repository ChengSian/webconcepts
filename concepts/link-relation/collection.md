**[RFC 6573: The Item and Collection Link Relations](/specs/IETF/RFC/6573 "RFC 5988 standardized a means of indicating the relationships between resources on the Web. This specification defines a pair of reciprocal link relation types that may be used to express the relationship between a collection and its members."):** [When included in a resource that represents a member of a collection, the 'collection' link relation identifies a target resource that represents a collection of which the context resource is a member.](http://tools.ietf.org/html/rfc6573#section-2.2)
