**[W3C TR http://www.w3.org/TR/preload: Preload](/specs/W3C/TR/preload "This specification defines the preload keyword that may be used with link elements. This keyword provides a declarative fetch primitive that initiates an early fetch and separates fetching from resource execution."):** [The preload keyword may be used with link elements. This keyword creates an external resource link (preload link) that is used to declare a resource and its fetch properties.](http://www.w3.org/TR/preload/#link-type-preload)
