Document Name: Internet Draft ietf-httpauth-mutual
Document URI: urn:ietf:id:ietf-httpauth-mutual
Online Version: http://tools.ietf.org/html/draft-ietf-httpauth-mutual
Organization: Internet Engineering Task Force (IETF)
Series: Internet Draft (I-D)
Abstract: This document specifies a mutual authentication scheme for the Hypertext Transfer Protocol (HTTP). This scheme provides true mutual authentication between an HTTP client and an HTTP server using password-based authentication. Unlike the Basic and Digest authentication schemes, the Mutual authentication scheme specified in this document assures the user that the server truly knows the user's encrypted password.
