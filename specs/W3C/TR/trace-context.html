Organization: World Wide Web Consortium (W3C)
Series: Technical Report (TR)
Abstract: This specification defines standard HTTP headers and a value format to propagate context information that enables distributed tracing scenarios. The specification standardizes how context information is sent and modified between services. Context information uniquely identifies individual requests in a distributed system and also defines a means to add and propagate provider-specific context information.


Specified Web Concepts:

HTTP Header Fields
